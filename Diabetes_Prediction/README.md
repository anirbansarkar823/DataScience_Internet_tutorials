- This project is about predicting whether the person (female) has diabetes or not. The target variable ('outcome'; 1-has diabetes: 0 - doesn't have diabetes) depends on below features:
	a. 'Pregnancies', 
	b. 'Glucose', 
	c. 'BloodPressure', 
	d. 'SkinThickness', 
	e. 'Insulin',
	f. 'BMI', 
	g. 'DiabetesPedigreeFunction', 
	h. 'Age', 
	i. 'Outcome'
